About me:

My name is Santiago Moliner, and I am a game developer. I’ve always wanted to create my own games, I love that feeling of accomplishment after finishing a project, seeing some else enjoying what I’ve created. You can check out some of my projects in here.

I specialize in coding, making it work efficiently and readable to other team members. In my free time I enjoy playing the piano, working out, and of course playing some video games.

Project_S(Alpha):

Project_S is a top down shooter (Twin stick) where the player has to survive rounds of enemies. The player has different weapons and abilities to choose from to defeat their enemies, and the game uses different high risk high reward mechanics.

Developed using the Unity engine with C# Scripts created from scratch. The game is still in its alpha stage, so there is much more to come.

Go to project repository!

Sorting Algorithm Visualized:

In this project i showcase some of the most used sorting algorithms used to order an array. I showcase step by step what is happening in the algorithm using different colors to represent the numbers that are being compared.

I used C++ and the library SDL for the visuals.

Go to project repository!

Sea Conquest:

Sea Conquest is a 2D Isometric RTS focused on exploration and resource management. The objective is to defeat the enemy pirates through strategy, base building and premeditated attacks.

In this project, I was the team leader of a 7 member team, were we had to develop an RTS using low level C++ using the libraries SDL for the visuals and STL.

Go to project repository!
